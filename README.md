## Usage

### 1、标注上报属性

#### 1.1、字段在DOM上分配说明：
| 标注DOM         |  说明                                   |  支持字段            |
| --------------- | --------------------------------------- | ---------------------|
| Body            |  document.body 上标注的属性              | 1、 td-groupid 可选  |
#### 1.2、支持的标注属性说明：

| 字段           | 示例                                   | 说明                                                         |
| -------------- | -------------------------------------- | ------------------------------------------------------------ |
| td-groupid     | td-groupid="front-end"                 | 1、页面的 body 标签上可以存在 groupid，不能放置在 body 除外的其它 dom 上|
